AI News Automation Agent (n8n + Gemini API)

Overview
This project is an AI-based automation workflow built using n8n. It collects news from different RSS feeds, processes the data, and generates short summaries using the Gemini API. The final output is sent as an email report.

Features

* Collects news from multiple RSS sources such as AI, Technology, and Defence
* Combines and processes data within the workflow
* Generates short summaries using Gemini API
* Sends automated email updates
* Can be scheduled to run at regular intervals

Tech Stack
n8n for workflow automation
Google Gemini API for summarization
RSS feeds as data sources

Workflow
The workflow starts by fetching data from RSS feeds.
The data is then merged and processed.
Summaries are generated using the Gemini API.
Finally, the results are sent as an email report.

How to Run
Import the workflow JSON file into n8n.
Add your Gemini API key.
Configure email settings.
Run the workflow or schedule it as needed.

Use Case
This project is useful for automating daily news updates. It saves time by providing quick summaries instead of reading full articles.
